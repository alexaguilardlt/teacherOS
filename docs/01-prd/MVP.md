# MVP - TeacherOS

Versión: 1.0

Estado: En diseño

Objetivo:

Validar que existe mercado para una herramienta que permita a un profesor generar y mantener automáticamente la planificación anual de una asignatura.

El MVP debe ser suficientemente útil como para que un profesor esté dispuesto a pagar una suscripción anual.

Todo lo que no contribuya a validar esta hipótesis queda fuera del MVP.

## Problema

Actualmente un profesor dedica muchas horas a:

- Crear la planificación anual.
- Reorganizar clases canceladas.
- Mantener actualizados varios documentos.
- Sincronizar diferentes herramientas.
- Repetir cada curso prácticamente el mismo trabajo.

La mayoría utiliza Excel porque no existe una herramienta sencilla que automatice este proceso.

## Hipótesis

Creemos que un profesor estará dispuesto a pagar una suscripción anual si consigue ahorrar varias horas de trabajo cada trimestre gracias a la planificación automática.

## Usuario principal

Profesor de secundaria.

Profesor de FP.

Profesor de academia.

Con conocimientos básicos de informática.

Que actualmente utilice Excel, Google Calendar o herramientas similares.

Un profesor debe poder crear toda la planificación de un curso en menos de 10 minutos.
